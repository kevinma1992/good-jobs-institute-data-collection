# Grocery Stores Locations Scraper for Good Jobs Institute Research Project (Fall 2020)
## Authors: Kevin Ma, Souhail Halaby, Aparna Pande
 This repository has the following folders:
 - employee-reviews: this folder contains code used to programmatically scrape employee reviews off of comparably.com.
 - google-reviews-ratings: this folder contains code used to programmatically scrape aggregate customer ratings for every grocery store's locations.
 - store-locators-scraper: this folder contains the code used to programmatically scrape locations off of each grocery store's store locator web page.
 - target-locations-list-cleaner: this folder contains the code used to transform and clean the Target locations list, which was found here: https://s7d2.scene7.com/is/content/Mattel/Hot%20Wheels%20Collector/01_Downloads/Target%20Promo/Hot%20Wheels%2050th%20-%20Store%20List.pdf.
