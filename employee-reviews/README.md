# Employee Reviews Scraper
## Authors: Kevin Ma, Souhail Halaby, Aparna Pande
 This folder contains code to scrape employee reviews data from comparably.com. This python script outputs a csv of aggregate employee ratings for each of the 18 dimensions on the website. There are no additional commands needed to run each script. To run the script, simply enter the following command into Terminal:

 python comparably_reviews.py
