# Grocery Stores Location Scraper
## Authors: Kevin Ma, Souhail Halaby, Aparna Pande
 This folder contains code to scrape locations data from store locator pages of grocery stores in our dataset. Each python script outputs a csv of the grocery store's location data; this output can be found in the 'output' folder. There are no additional commands needed to run each script. To run each script, simply enter the following command into Terminal:

 python [script_name].py
